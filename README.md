**Walmart Sales Analysis using SQL**



\*Project Overview:

This project analyzes Walmart sales data using SQL to uncover business insights and sales trends. The analysis focuses on revenue performance, product categories, customer behavior, and branch-level sales performance.



\*Tools Used:

\- SQL

\- Excel

\- GitHub



\*Dataset:

The dataset contains Walmart sales transactions including:

\- Invoice ID

\- Branch

\- City

\- Product Line

\- Quantity

\- Total Sales

\- Payment Method

\- Customer Type

\- Rating



\*Business Questions Solved:

1\. Which branch generates the highest revenue?

2\. Which product line has the highest sales?

3\. What are the most common payment methods?

4\. Which city contributes the most sales?

5\. What are the peak sales periods?

6\. Which customer type generates more revenue?



\*Key Insights:

Detailed insights are available in:

\- walmart analysis insights.pdf

\- Business Insights.txt





\*Files Included:

\- Walmart Sales Data.xlsx

\- walmart\_sales\_analysis.sql

\- walmart analysis insights.pdf

\- Business Insights.txt



\*Project Outcome:

The analysis helped identify top-performing branches, product categories, customer purchasing patterns, and revenue trends that can support business decision-making.
